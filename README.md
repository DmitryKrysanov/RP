# RP Live. Lesson 3 - Устанавливаем Node сервер

## Установка

1) Склонируйте этот репозиторий

    ```
    $ git clone -b lesson-3 https://github.com/Ureth/RP.git

    ```

2) Выполните в командной строке

    ```
    $ cd RP
    $ npm install

    ```
